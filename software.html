Software

iSoLIM

iSoLIM (intelligent Soil-Landscape Inference Model) is a software implementation of the SoLIM framework designed for the computation demand in the big data era. The "intelligent" is currently achieved from the aspect of data processing. It support efficient computation on large gridded raster data on personal computer, and allows for integration of soil mapping knowledge from multiple sources. It runs as a Microsoft Windows executable, Click here to obtain the software and associated material (user manual and tutorial data). Click here to obtain the user manual.

CyberSoLIM

Back-ended with SoLIM, CyberSoLIM provides a heuristically driven, visually assisted, and high performance computing enabled cyber environment for digital soil mapping (DSM). Follow this link to get started with CyberSoLIM. Use the help menu on the upright corner of the screen after you have logon to the software system.

SoLIM Software

The SoLIMSolutions software contains a set of tools for predictive digital soil mapping under the SoLIM framework and is available for non-commercial use. It contains the improvements over the earlier versions (SoLIM Suite and SoLIM software). Much of the work was funded by the Chinese Academy of Sciences and the USDA Natural Resources Conservation Service. It runs as a Microsoft Windows executable. Follow this link to obtain the software and associated materials (The Functionality Manual and Tutorial and tutorial datasets).
